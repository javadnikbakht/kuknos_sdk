********
异步请求
********

现在我们支持通过异步请求来提交事务了，当然我们不会强迫你使用异步，毕竟同步代码更容易编写，也有着更多的开发库可以选择。

:py:class:`Server <kuknos_sdk.server.Server>` 有一个参数是 **client**，在这里我们有必要提及一下它，如果你不明确的设置这个参数，
那么 SDK 将默认使用 :class:`RequestsClient <kuknos_sdk.client.requests_client.RequestsClient>` 的实例，它是一个同步的客户端；
当然，你也可以指定一个异步的 HTTP 客户端，例如： :class:`AiohttpClient <kuknos_sdk.client.aiohttp_client.AiohttpClient>`。
如果你设置了一个异步的客户端，那么所有的请求都是异步的，反之则是同步的。
